## Como utilizar este endpoint?

Esse endpoint ativa varios aplicativos dentro da plataforma .

### Headers obrigatórios

<ParamField
    header="Authorization"
    type="string"
    placeholder="Bearer {token}"
    required
>
    O token de autenticação no formato Bearer é obrigatório para acessar este
    endpoint.
</ParamField>

### Exemplo de Requisição

```json
POST https://mcapi.knewcms.com:2087/lines/active/app/xstream
Headers:
{
  "Authorization": "Bearer seu_token_aqui",
  "Content-Type": "application/json"
}
Payload:
{
    "nameApp": "Kplay",
    "mac": "K4S5S5",
    "namePlaylist": "test",
    "id_user": 102313925
}
```

### Parâmetros do corpo da requisição (Payload)

<ParamField body="nameApp" type="string" placeholder="123456" required>
    Nome do aplicativo a ser ativado. OPções disponíveis:
    WTV Player/Wapp, XCloud, Kplay
</ParamField>

<ParamField body="mac" type="string" placeholder="SD2DG" required>
    Código do cliente
</ParamField>

<ParamField body="namePlaylist" type="string" placeholder="teste" required>
    Nome da playlist a ser ativada.
</ParamField>

<ParamField body="id_user" type="number" placeholder="12333" required>
    Id do cliente a ser ativado.
</ParamField>

<ResponseExample>
    ```json
    {
        "success": true
    }
    ```
</ResponseExample>
