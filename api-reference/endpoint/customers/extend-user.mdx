## Como utilizar este endpoint?

Esse endpoint estende o tempo de um **usuário** já existente, adicionando créditos ao mesmo.

### Headers obrigatórios

<ParamField
  header="Authorization"
  type="string"
  placeholder="Bearer {token}"
  required
>
  O token de autenticação no formato Bearer é obrigatório para acessar este endpoint.
</ParamField>

### Parâmetros do corpo da requisição (Payload)

<ParamField body="credits" type="number" placeholder="Quantidade de créditos" required>
  Quantidade de créditos a serem adicionados ao usuário.
</ParamField>

### Exemplo de Requisição

```json
POST https://mcapi.knewcms.com:2087/lines/extend/{id}
Headers:
{
  "Authorization": "Bearer seu_token_aqui",
  "Content-Type": "application/json"
}
Payload:
{
    "credits": 1
}
```
