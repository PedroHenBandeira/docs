## Como utilizar este endpoint?

Esse endpoint ativa varios aplicativos dentro da plataforma .

### Headers obrigatórios

<ParamField
  header="Authorization"
  type="string"
  placeholder="Bearer {token}"
  required
>
  O token de autenticação no formato Bearer é obrigatório para acessar este
  endpoint.
</ParamField>

### Exemplo de Requisição

```json
POST https://mcapi.knewcms.com:2087/lines/active/app
Headers:
{
  "Authorization": "Bearer seu_token_aqui",
  "Content-Type": "application/json"
}
Payload:
{
    "nameApp": "Wapp",
    "mac": "K4S5S5",
    "namePlaylist": "test",
    "id_user": 102313925
}
```

### Resposta

<ResponseField name="status" type="number">
  Indica o código HTTP da resposta. Esperado: 201 (Created).
</ResponseField>
